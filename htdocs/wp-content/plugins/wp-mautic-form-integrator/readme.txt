== Description ==
### Introduction
[Mautic](https://www.mautic.org/) is an open-source marketing automation software and WP Mautic Form Integrator plugin is a bridge between Mautic’s back-end interface and many of the most frequently used form builder plugins in WordPress.

WP Mautic Form Integrator will allow you to use the form builder plugin you are already using to collect email data and other information for Mautic’s marketing automation. Without writing a single line of code you can bring your email lists into one of the most comprehensive open-source marketing applications available.

With WP Mautic Form Integrator plugin you are able to map the form fields of your chosen form builder plugin with the fields defined within Mautic. To use, just install the WP Mautic Form Integrator plugin, set up the value you wish to collect, connect them with the pre-existing Mautic values, and you will be ready to integrate the forms in no time.

### Key features
* Easy setup and integration
* Store data from your WordPress contact plugin in Mautic
* Simple to use UI to map the form fields with the Mautic form fields.
* Keep the data map even if you delete or deactivate the plugin

### Compatibility
WP Mautic Form Integrator plugin currently works with the following plugins;

Open source plugins

* [Contact Form 7](https://wordpress.org/plugins/contact-form-7/)
* [Fast Secure Forms](https://wordpress.org/plugins/si-contact-form/)
* [Ninja Forms](https://wordpress.org/plugins/ninja-forms/)
* [Formidable Forms](https://wordpress.org/plugins/formidable/)

Premium plugins

* [Gravity Forms](http://www.gravityforms.com/)

== Installation ==
#### Automatic Installation
1. Visit Plugins > Add New
1. Search for WP Mautic Form Integrator
1. Activate WP Mautic Form Integrator from your Plugins page.

#### Manual Installation
1. Download WP Mautic Form Integrator Plugin.
1. Upload the wp-mautic-form-integrator directory to your /wp-content/plugins/ directory, using your favorite method (ftp, sftp, scp)
1. Activate WP Mautic Form Integrator from your Plugins page.

#### Manual Installation
1. Download WP Mautic Form Integrator Plugin.
1. Visit Plugins > Add New
1. Click on upload plugin button.
1. Click on browse button and select plugin zip file. After file select click on Install Now button.
1. Click on Activate Plugin button.

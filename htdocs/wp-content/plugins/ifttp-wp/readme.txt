Like the popular tool IFTTT, this plugin allows you to trigger conditional actions on posts based on tags.

== Description ==

Ideal for pairing with the WordPress mobile app, this plugin lets you set set trigger conditional actions on posts based on post_tags added to the post. For this initial version, you can change the post type. More functionality and actions planned.

== Installation ==

1. Upload to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Set up actions under the menu Settings > IFTTP Settings

== Frequently Asked Questions ==

= Why would I want to use this, can't I do it from the app already? =

The WordPress app does not support custom post types, I created this so that you can set posts to the desired post type from the app.
